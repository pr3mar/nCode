Цезаровата шифра е еден од најстарите начини на шифрирање на било кој текст. Се нарекува цезарова затоа што императорот Јулие Цезар ја користел за да комуницира со своите команданти, со цел побезбедно да ги пренесува своите пораки коишто содржеле воена содржина. Во тоа време овој начин бил толку безбеден што оние кои ќе ја украделе пораката со цел да ја прочитаат мислеле дека е напишано на некој странски јазик и само ја занемарувале пораката која всушност имала големо значење.

Магијата на овој начин на шифрирање се состои во поместување на буквите налево или надесно за одреден број на букви. Што значи доколку имаме порака „ЗДРАВО“ би била шифрирана како „ЃБЊЧШМ“. Или поконкретно на слика 3 прикажано:

Многу се поедноставува разбирањето на ова шифрирање со помош на математичкиот модел кој гласи:

  1. E_n (x) = (x + n) mod 26 – формула за шифрирање на текстот
  2. D_n (x) = (x - n) mod 26 – формула за дешифрирање на текстот
  3. x – позицијата на буквата во абецедата
  4. n – поместување на буквата
  5. mod 26 се користи за да не се преминува преку границата од 26 букви, колку што има во абецедата, и кога ќе се премине да се почне од првата буква повторно.

Дел од кодот што ја извршува оваа работа е:

    // функција за шифрирање на порака со помош на методот на цезаровата шифра
    function cezarova_sifra($poraka, $pomestuvanje)
    {
        $sifra = '';
        $length = strlen($poraka);
        // шифрирај го секој карактер од пораката
        for($i = 0; $i < $length; $i++)
        {
            // најди ја позицијата на буквата во абецедата
            $check = ord($poraka[$i]) - ord('a')-1;
            // пресметај ја новата поцизија
            $shifted = ($check + $pomestuvanje) % 26;//key
            // додај го карактерот на шифрираната порака
            $sifra .= chr($shifted + ord('a')-1);
        }
        // врати ја добиената вредност
        return $sifra;
    }
